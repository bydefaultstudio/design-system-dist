# design-system-dist

Build artefacts of the By Default design system, published automatically from the private source repository on each release tag. The only contents are `dist/design-system.css` and a minimal `package.json`.

## Install

```bash
npm install github:bydefaultstudio/design-system-dist#semver:^1.0.2
```

Releases are tagged `v*` and match the source repository's package version.
